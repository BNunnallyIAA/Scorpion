Downloading Scorpion

Here you can get the latest Scorpion distributions for you device. The current stable version is 0.0.19_19.

Ways of Installing Scorpion

Depending on your device and operating system, the ways to install Scorpion may vary. If you have Linux or Mac, try running this in your console:

$ cd your_root_dir/Github
$ git clone https://github.com/AndroDevcd/Scorpion/
$ cd Scorpion/
$ sudo cp "bin/linux_debian/raspberryPi/0.0.19_19 beta" /bin
$ sudo scorpion -help
